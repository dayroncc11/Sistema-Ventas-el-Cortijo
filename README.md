# Sistema de Ventas El Cortijo

Sistema web desarrollado con Spring Boot y Java para la gestión de ventas, clientes y productos.

## Tecnologías

- Java
- Spring Boot
- Maven
- Spring Data JPA
- Thymeleaf
- MySQL
